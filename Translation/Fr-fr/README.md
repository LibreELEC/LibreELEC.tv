# LibreELEC

LibreELEC ist ein 'gerade genug OS' Linux Distribution für die preisgekrönte [Kodi] (https://kodi.tv) Software auf beliebten Mediacenter Hardware. LibreELEC ist eine konservative Gabel der beliebten [OpenELEC] (http://openelec.tv) Projekt mit einem stärkeren Fokus auf Pre-release testen und post-release Change Management. Weitere Informationen über das Projekt können Sie auf der [Website] (LibreELEC Https://libreelec.tv). gefunden werden
